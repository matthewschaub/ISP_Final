View Grades

Actions

Display all grades: Type in the ID of the student, and all quizzes and grades for that student will be displayed

Add a new test grade: Type in the ID of the student, the name of the test and their grade for the test, and it
will be added to the students scores

Update an existing student grade: Type in the ID of the student, and the name of the test and the new grade for the test.

Delete an existing test grade: Type in the ID of the student and the name of the test you wish to delete
